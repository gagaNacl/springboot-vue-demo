# Vue 3 Admin Template

一个基于 Vue 3 + TypeScript + Vite + Element Plus 的前端管理系统模板

## 项目特性

- 💪 Vue 3.3 + TypeScript
- 🔥 使用 Vite 4.x 作为构建工具
- 🎨 Element Plus UI 组件库
- 📦 Pinia 状态管理
- 🔒 基于 JWT 的用户认证
- 📝 使用 Composition API 和 `<script setup>` 语法
- 🎯 响应式布局，支持多端适配
- 🌐 封装 Axios 请求
- 🚦 权限路由控制
- 📚 多环境配置
